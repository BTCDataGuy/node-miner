# Gallery Screenshots

This folder should contain 3 screenshots of the Node Miner app for the Umbrel App Store.

## Requirements

- 3 images named: `1.jpg`, `2.jpg`, `3.jpg`
- Resolution: 1920x1080 or 1280x720
- Format: JPG

## Suggested Screenshots

1. **Dashboard** - Show the mining control panel with gauges
2. **Pool Settings** - Show the configuration interface
3. **Terminal** - Show the miner output

## How to create

1. Run the app in Umbrel
2. Take screenshots of different pages
3. Resize to 1920x1080 or 1280x720
4. Save as `1.jpg`, `2.jpg`, `3.jpg` in this folder
5. Delete this README.md
